## 📁 含まれているテスト
- `google_search.py`: Google検索自動化テスト
- `login_test/`: ログインフォームの正常・異常テスト

## 📚 学び・気づき（Git/GitHub・構成整理）

  - 今回は少数の小さなテストコードなので同リポジトリで管理

- GitHubで構成が崩れたとき、落ち着いて対応すること、構造と履歴を意識して整理し直す力が大切だと実感
